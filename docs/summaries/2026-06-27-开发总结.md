## 关键技术点

1. **路径分离**：`code/` 存放所有工程代码，`config/`/`data/`/`logs/` 在项目根目录，通过 `config.py` 中的 `PROJECT_ROOT` 常量统一管理
2. **JSON 字段透明化**：Profile 的 JSON 字段由 DAO 层自动序列化/反序列化，上层代码只操作 Python dict/list
3. **分页查询**：`get_recent` 取最近 N 条升序，`get_before` 取指定时间戳之前 N 条升序。实现用 "DESC + LIMIT + reverse" 模式保证性能
4. **测试隔离**：每个测试用例使用独立临时 SQLite 数据库，通过 monkey-patch `config.DB_PATH` 实现路径重定向
5. **WAL 模式**：SQLite 启用 WAL（Write-Ahead Logging），提升并发读写性能
6. **QListWidget + setItemWidget**：聊天区使用 QListWidgetItem 嵌入自定义 MessageBubble，支持滚动分页。比 QScrollArea+VBoxLayout 更轻量
7. **无框模态弹窗**：StrategyDialog 使用 `FramelessWindowHint` + 透明背景 + 橙色边框内层，实现带边框的模态效果
