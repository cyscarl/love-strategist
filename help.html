聊天助手 使用帮助

本软件是一个聊天辅助工具。它通过人工智能分析你和对方的聊天记录，帮你生成高情商的回复建议，推进人际关系。

简单来说：你录入聊天内容 → AI 分析对方性格和情绪 → 给你 3 条回复建议 → 你选一条发送。

第一步：配置 AI 服务

1. 点击左侧导航栏的 「设置」。

2. 在「供应商」下拉框中选择你使用的 AI 服务商（如 DeepSeek、OpenAI）。

3. 填写你的 API Key（密钥）。这是从 AI 服务商获得的，类似一串密码。

4. 模型会自动填充，你也可以点击右边的 ⟳ 按钮刷新列表。

5. 点击 「保存配置」，然后点击 「测试连接」。如果看到"连接成功"，说明配置正确。

在哪里获取 API Key？
· DeepSeek：访问 platform.deepseek.com → 注册 → API Keys → 创建
· OpenAI：访问 platform.openai.com → 注册 → API Keys → 创建
注意：API 服务通常是付费的，需要充值才能使用。

第二步：创建联系人

1. 点击左侧导航栏的 「通讯录」。

2. 点击左侧列表上方的 「新建联系人」。

3. 输入对方的名字，点击确定。

4. 在右侧可以设置头像、添加备注。

「自己」的联系人：通讯录第一个是你自己。可以修改自己的名字和头像，你在聊天中的头像和名字会同步更新。

第三步：录入聊天记录

1. 点击左侧导航栏的 「聊天」，在中间列表选择联系人。

2. 在底部输入框中输入消息，按 Enter 键发送。

3. 己方消息（你说的）：直接输入后按 Enter，或点击绿色「发送」按钮。

4. 对方消息（对方说的）：点击白色「对方发送」按钮，输入内容后确认。

提示：你可以手动输入双方的聊天记录，就像你在微信里看到的对话一样。 聊天气泡在右侧是你说的话，在左侧是对方说的话。

第四步：使用 AI 助手

1. 在聊天界面，点击底部按钮栏中的 「求助」按钮。

2. 等待几秒，AI 会分析你们的对话内容，生成 3 条回复建议。

3. 建议会显示在输入框上方。点击任意一条即可填入输入框。

4. 如果不满意，点击 ⟳（刷新）按钮重新生成。

5. 如果当前情况特殊，点击 「策略」按钮，告诉 AI 你的想法（如"她不太开心"），AI 会据此调整。

查看人物画像

切换到 「通讯录」页面，点击任意联系人。

右侧会显示：

  • 好感度：AI 根据聊天内容估算的好感度分数（-100 到 100）
  • 人格摘要：AI 分析的对方性格特点、兴趣偏好等
  • 备注：你可以添加私人笔记，不会影响 AI 判断

点击右上角 「...」→ 「查看详情」可以打开完整的画像报告。

在画像弹窗中点击 「重新分析」可以让 AI 重新分析最新对话。

聊天消息操作

在聊天记录上 右键单击，可以：

  • 复制：将消息文本复制到剪贴板
  • 编辑内容：修改这条消息的文字
  • 删除：删除这条消息
  • 插入消息：在这条消息上面插入一条新的己方或对方消息

常见问题

Q：点击求助后没有反应？
A：请确认已在设置中正确配置 API Key，并点击"测试连接"确认可用。
Q：为什么 AI 生成的回复不太合适？
A：可以点击「策略」按钮，告诉 AI 你的具体需求。比如"她今天情绪不好，帮我安慰她"或者"我想约她周末出去玩，帮我找个自然的方式提起"。
Q：好感度为什么不准确？
A：好感度是 AI 根据聊天内容估算的，仅供参考。你可以手动编辑画像数据来调整。
Q：软件数据存储在哪里？
A：所有数据存储在软件目录下的 data/ 文件夹中。日志文件在 logs/ 文件夹中。可以在设置中打开日志目录。
